
# Farsava - Health Api

First create an `API KEY` [here](https://panel.amerandish.com/)

## install dependencies

```
dotnet restore
```

## configs
```c#
string baseUrl = "https://api.amerandish.com/v1";
string actionUrl = "/speech/asr";
string authKey = "<YOUR_API_KEY>";
```

## run

```bash
dotnet run
```
